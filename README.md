# SAP_LAB5_Space_Invaders
Это PyGame-порт Space Invaders с той же логикой, что и JavaScript-версия. Класс Game управляет циклом на 60 FPS через pygame.time.Clock() и хранит объекты в списке bodies. Invader патрулирует, меняя направление при patrolX > 500, и стреляет с шансом 2%, если под ним нет других врагов. Player управляется клавишами K_LEFT/K_RIGHT/K_SPACE с лимитом 5 пуль и перезарядкой каждые 12 кадров. Bullet просто двигается по скорости.
В методе update() применяется двойная фильтрация: сначала удаляются все объекты, которые сталкиваются с любыми другими (проверка AABB-коллизий через all() и colliding()), затем удаляются объекты, вышедшие за вертикальные границы экрана.
Отрисовка в draw() использует цветные прямоугольники: игрок — зелёный, пришельцы — красные, пули — синие. Функция draw_body() унифицирует отрисовку через pygame.draw.rect()
Игра запускается через main(), создаёт окно 800×600 и запускает игровой цикл run(). Логика полностью сохраняет оригинальную механику.
